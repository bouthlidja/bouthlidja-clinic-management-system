# Clinic Management System

## 📌 Project Description
A web-based system for managing clinic operations including appointments, patients, doctors, and medical records.

## 📚 Documentation
This project follows a documentation-first approach using Software Requirements Specification (SRS).

## 🛠️ Technologies
- HTML, CSS, JavaScript (Vue.js)
- PHP (Backend)
- MySQL Database
- LaTeX (SRS Documentation)

## 📁 Project Structure
- docs/ → SRS documentation
- frontend/ → Web interface
- backend/ → Server-side code
